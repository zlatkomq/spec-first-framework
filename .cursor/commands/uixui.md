# uixui

Create UIX-UI.md from an approved SPEC and DESIGN.

Apply @.cursor/rules/uix-ui-creation.mdc using @.framework/templates/UIX-UI.template.md and @.framework/CONSTITUTION.md.

The user's message after /uixui should reference the spec (e.g. "006" or "@specs/006-user-export/DESIGN.md"). If not provided, ask which spec to create UIX/UI specs for.

If the user provides Figma URLs, use Figma MCP tools to extract design data and incorporate it into the UIX-UI.md artifact.
