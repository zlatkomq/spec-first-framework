# uix

Create UIX-SPEC.md (Figma mapping) from an approved DESIGN.

Apply @.cursor/rules/uix-creation.mdc using @.framework/templates/UIX-SPEC.template.md.

The user's message after /uix should reference the spec (e.g. "006" or "@specs/006-user-export/DESIGN.md"). If not provided, ask which spec to create a UIX mapping for. Both SPEC.md and DESIGN.md must be APPROVED before proceeding.
